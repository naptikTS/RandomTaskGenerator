# Expense Tracker - Трекер расходов

**Автор:** [Илья Ткачев]

## Описание программы
Expense Tracker - это приложение для отслеживания личных расходов с графическим интерфейсом. Позволяет добавлять расходы, фильтровать по категориям и датам, а также подсчитывать общую сумму за выбранный период.

## Функциональность
- ✅ Добавление расходов с указанием суммы, категории и даты
- ✅ Выбор категории из списка (Еда, Транспорт, Развлечения, Жильё, Здоровье, Одежда, Связь, Другое)
- ✅ Подсчёт суммы расходов за выбранный период
- ✅ Фильтрация по категории
- ✅ Фильтрация по диапазону дат
- ✅ Сохранение и загрузка данных из JSON файла
- ✅ Валидация ввода:
  - Сумма должна быть положительным числом
  - Дата в формате ГГГГ-ММ-ДД
- ✅ Удаление отдельных записей
- ✅ Очистка всей истории

## Технологии
- Python 3.6+
- Tkinter (GUI)
- JSON (хранение данных)
