# Training Planner - Планировщик тренировок

**Автор:** [Твоё имя и фамилия]

## Описание проекта
Приложение для планирования тренировок с графическим интерфейсом. Позволяет добавлять, просматривать, фильтровать и удалять тренировки.

## Функциональность
- ✅ Добавление тренировок с указанием даты, типа и длительности
- ✅ Валидация даты (формат ГГГГ-ММ-ДД)
- ✅ Валидация длительности (положительное число)
- ✅ Фильтрация по диапазону дат
- ✅ Фильтрация по типу тренировки
- ✅ Сохранение данных в JSON файл
- ✅ Загрузка данных из JSON при запуске
- ✅ Удаление тренировок

## Технологии
- Python 3.6+
- Tkinter (GUI)
- JSON (хранение данных)
