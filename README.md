# Password Generator - Генератор паролей

**Автор:** [Илья Ткачёв]

## Описание проекта
Приложение для генерации случайных паролей с настраиваемыми параметрами. Позволяет выбрать длину пароля и типы используемых символов, сохраняет историю сгенерированных паролей.

## Функциональность
- ✅ Ползунок для выбора длины пароля (4-32 символа)
- ✅ Чекбоксы для выбора типов символов:
  - Заглавные буквы (A-Z)
  - Строчные буквы (a-z)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&*)
- ✅ Кнопка генерации пароля
- ✅ Отображение сгенерированного пароля
- ✅ Копирование пароля в буфер обмена
- ✅ История всех сгенерированных паролей
- ✅ Оценка сложности пароля (Слабый/Средний/Сильный)
- ✅ Сохранение истории в JSON файл
- ✅ Валидация ввода:
  - Проверка выбора хотя бы одного типа символов
  - Проверка длины пароля (минимум 4, максимум 32)

## Технологии
- Python 3.6+
- Tkinter (GUI)
- JSON (хранение истории)
- Random (генерация паролей)
